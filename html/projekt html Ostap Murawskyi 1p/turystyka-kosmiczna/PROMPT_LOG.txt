INSTRUKCJE GENEROWANIA GRAFIK:
================================

1. Użyć DALL-E 3 (premium) lub Bing Image Creator (darmowy) lub Leonardo.ai
2. Dla każdego promptu:
   - Kliknąć "Generate" lub "Create"
   - Poczekać 30-60 sekund
   - Pobrać obraz w najwyższej dostępnej rozdzielczości
   - Zapisać jako JPG w folderze images/

3. Kryteria akceptacji:
   ✓ Realistyczne, wysokiej jakości
   ✓ Spójne kolorystycznie
   ✓ Odpowiadające tematyce
   ✓ Bez tekstu na obrazach
   ✓ Wymiary odpowiadające planom

4. Jeśli pierwszy wynik nie spodaba się:
   - Regenerować z tym samym promptem
   - Lub dostosować prompt i spróbować ponownie
   - Zazwyczaj trzeba 2-3 próby aby uzyskać doskonały wynik
