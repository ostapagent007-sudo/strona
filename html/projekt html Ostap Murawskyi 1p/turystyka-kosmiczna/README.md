## 🎨 Jak Generować Grafiki AI

### Narzędzia (wybierz jeden):
1. **DALL-E 3** (Premium, najlepszej jakości)
   - https://openai.com/dall-e-3
   - Wymaga OpenAI account

2. **Bing Image Creator** (Darmowy, dobre rezultaty)
   - https://www.bing.com/create
   - Wymaga Microsoft account

3. **Leonardo.ai** (Darmowy, szybki)
   - https://leonardo.ai
   - Wymaga free account

### Krok po Kroku:

1. Otwórz wybrany generator obrazów
2. Skopiuj prompt z PROMPT_LOG.txt
3. Kliknij "Generate" / "Create"
4. Poczekaj ~30-60 sekund
5. Kliknij "Download" / "Save"
6. Wybierz najwyższą rozdzielczość (minimum 1080p)
7. Zapisz plik jako **JPG** w lokalizacji: `images/nazwa.jpg`

## 💡 Podpowiedzi do Zrozumienia Kodu

### Nawigacja `<nav>`
```html
<nav>
    <a href="index.html">Strona główna</a>
    <a href="o-nas.html">O nas</a>
</nav>
```
- To pozwala przechodzić między stronami
- Na każdej stronie menu jest identyczne
- Każdy `<a href="">` wskazuje do innego pliku

### Tabela `<table>`
```html
<table>
    <tr><th>Nazwa</th><th>Cena</th></tr>
    <tr><td>Pakiet A</td><td>1000 zł</td></tr>
</table>
```
- `<table>` = cała tabela
- `<tr>` = wiersz (row)
- `<td>` = komórka danych (data)
- `<th>` = nagłówek kolumny

### Formatowanie Tekstu
```html
<h1>Nagłówek 1 - Największy</h1>
<h2>Nagłówek 2</h2>
<h3>Nagłówek 3 - Najmniejszy</h3>
<strong>Pogrubienie</strong>
<ul><li>Lista punktowana</li></ul>
<ol><li>Lista numerowana</li></ol>
```

## ❓ FAQ

**Q:** Czy muszę programować nowy kod?  
**A:** NIE! Wszystkie strony HTML i CSS są już kopiuj-wklej gotowe. Musisz tylko dodać grafiki AI.

**Q:** Ile grafik naprawdę potrzebuję?  
**A:** Minimum 5. Masz 10 zaplanowanych, ale 5 wystarczy.

**Q:** Czy mogę zmienić kolory CSS?  
**A:** TAK! Ale czarne + fioletowe + złoto są już spójne. Jeśli chcesz zmienić, zmień konsekwentnie.

**Q:** Co jeśli nie będę mieć wszystkich grafik?  
**A:** Może być HTML bez grafik - każdy `<img>` pokazze placeholder. Ale lepiej mieć grafiki.

**Q:** Czy mogę zmienić treść tekstu?  
**A:** TAK! Własna kreatywność = bonus punktów!
